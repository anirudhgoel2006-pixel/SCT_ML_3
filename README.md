Dogs vs Cats Image Classification

Overview

This project implements an image classification model that distinguishes between dogs and cats using machine learning and computer vision techniques.

Features

- Binary image classification (Dog vs Cat)
- Image preprocessing and resizing
- Model training and evaluation
- Prediction on new images
- Organized project structure

Project Structure

SCT_ML_3/
├── config/
├── dataset/
│   ├── cats/
│   └── dogs/
├── src/
├── tests/
├── .gitignore
├── README.md
├── requirements.txt
└── run.py

Dataset

The dataset contains images of cats and dogs used for training and testing the classification model.

Installation

pip install -r requirements.txt

Run the Project

python run.py

Technologies Used

- Python
- NumPy
- OpenCV
- TensorFlow / Keras
- Scikit-learn

Results

The model is trained to classify images as either a dog or a cat and can be used for prediction on unseen images.
